# Temporal CLI binaries

## Instructions

* Create folders in this directory named after the version of the temporal-cli binary version.

* [Temporal-CLI Binaries](https://github.com/temporalio/cli/releases)

* Use the `linux_amd64` binaries

```text
binaries
  |
  |-> 1.5.1
  |     |-> temporal
  |
  |-> 1.6.1
        |-> temporal
```
